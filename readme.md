It's simple project to see gifs from giphy.com

Stack:
Clean architecture
Modularization
MVVM
Dependency injection (Hilt)
Retrofit2
Glide
Paging 3
Room
Navigation component
Single activity

How to use:
Put your API key into local.properties file like this API_KEY=<your key>
